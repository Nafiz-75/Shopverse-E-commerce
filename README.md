# Shopverse-E-commerce
A fully responsive, single-page e-commerce front-end built with HTML, Tailwind CSS, and vanilla JavaScript. Features product listings, category filtering (Fashion, Electronics, Groceries), and a real-time 'Add to Cart' notification system.
